pxAnima beginner's guide

1) Load video file or an image

Press load media button to load a media file. Main supported format is mp4 but you're welcome to try others. While this program is meant for videos, images work as well. In this example we will be using drone footage provided in the example folder.

2) Color palette

There is no particular order you have to manipulate any of the sliders or settings. I like to start with a color palette. I decided to use 'Firestorm'.

You can remove colors with right click and edit by left click. To add a new one, press the orange (+) button.

3) Playing with sliders

Just like the title. Each video requires different approach. Did I mention that it'd be best if you went out into the real world and shot your own video? If you plan to use this for character animation reference, using a simple one color background helps. Don't wear white clothes when the wall behind you is also white :)

4) Video playback options

There are 3 handles on the video playback bar. The central one shows the current timestamp. With left and right one you can limit the playback range and only render the part of the video that's interesting to you.

On the right side of the video there are a few simple playback and recording options. Playback ones are pretty self explanatory

Recording fps slider is exactly that. How many frames per a second of playback the application will attempt to save.

Scale is a boolean, whether to save 1pixel to 1 pixel in the file or 1 pixel to many pixel squares (pixelizer size). I know, ugly solution. No time though :)

5) Recording

Record button is a toggle. When you press it the video will jump to the leftmost slider handle (start of the loop) and will try to keep saving .png images to the provided folder until you turn it off. I recommend to let it run at least twice. It's yet another ugly solution to a core freature that I didnt have time to expand on. Making this more user friendly and consistent is a top priority in the coming days.
